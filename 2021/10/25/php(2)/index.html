    ?>

    • count() 里面加数组名称，返回元素的数量

  • 关联数组 (类似STL中的map)

    • 遍历关联数组 foreach

      foreach($age as $x=>$x_value)
      {
      echo $x $x_value; // key to value
      echo “< br>”;
      }

  • sort() 里面跟数组名，升序排序

    rsort() 降序排序

    asort() - 根据关联数组的值，对数组进行升序排列

    ksort() - 根据关联数组的键，对数组进行升序排列

    arsort() - 根据关联数组的值，对数组进行降序排列

    krsort() - 根据关联数组的键，对数组进行降序排列

    __ LINE__的值为他所在的行数

    __ FILE__ 是一个绝对路径

    __ FUNCTION__返回该函数被定义时的名字
